# VPN和代理服务器的区别  
代理服务器和虚拟专用网（VPN）具有相同的用途，两者都旨在在线隐藏用户的身份，并且允许你更安全地访问互联网上的内容。同时，也可以使用任何一个虚拟位置为自己创建网络标识。例如，当你坐在洛杉矶的咖啡厅，你可以使用代理服务器通过澳大利亚的服务器连接到互联网，这样一来，网友就会认为你人在澳大利亚了。  

要理解两者之间的区别，首先需要它们的作用搞明白。

**什么是代理服务器？**  

代理服务器充当着你的电脑和另一台计算机之间的联络人，当你使用代理连接到互联网时，你需要通过代理服务器请求连接。当你在计算机上执行的所有操作也称为客户端，将转发到代理服务器，该代理收集你的要求信息，随之将其返送给你。  

这样一来，代理就可以在线隐藏你的身份，你使用代理服务器作为掩码，它将会代替你，并代表你做事。  

<br/>


**什么是VPN？**  

VPN或虚拟 专用网络允许你将计算机连接到远程网络，VPN通常专门用于连接到本地intranet或外部链接不可访问的内容。许多工作场所允许他们的员工使用VPN远程连接，以便，以便从办公室外部访问公司的本地Intranet上的数据。

**有什么不同？**  

(1)连接的协议不同 vpn是PTTP（点对点隧道协议），L2TP（第2层隧道协议）等。而代理服务器是ss  ssr v2ray clash等协议    
(2)在虚拟专用网络的情况下，客户端的IP地址是可见的，并且不会通过网络隐藏。而代理服务器在客户端的代理IP地址被隐藏并被网络隐藏的情况下，使用匿名网络ID作为IP  
(3)虚拟专用网络提供了较高的安全性，并且与代理相比更加安全。而尽管Proxy不能提供与VPN一样安全的连接，因此与VPN相比，其安全性较低。
